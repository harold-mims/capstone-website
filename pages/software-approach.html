Software Descriptions
Flutter
Companion App Our cross-platform companion application is developed using the framework Flutter, ensuring a seamless and user-friendly experience while also providing invaluable debugging information for streamlined development. Leveraging the power of the Flutter_Blue_Plus package, our app monitors the Bluetooth device status, automatically scanning for the HM-11 module used with our hardware. Once a connection is established, the main screen empowers the user to modify the values utilized by the embedded system for precise calculation of the STEP signal required by the stepper motor. With a simple and intuitive interface, users have the flexibility to enable/disable and adjust the speed within the range of 0-100% with utmost ease and control.

A more detailed breakdown can be found at our github.

C
Embedded System The embedded system developed for the MSP430F45994 microcontroller leverages UART communication to interface with the HM-11 Bluetooth module. The system uses the UART protocol to establish a bidirectional communication link with a Bluetooth-enabled device. Incoming data from the Bluetooth module is parsed and processed to extract control commands for the stepper motor. The embedded system utilizes these commands to dynamically adjust the stepper motor's speed while limit switches change the stepper motor's direction. This integration of Bluetooth and stepper motor control offers a versatile and efficient solution for wireless control.

A more detailed breakdown can be found at our github.
